# Titanic-Kaggle-Competition
# Titanic Exploratory Data Analysis (EDA)

## Overview

This repository contains my Exploratory Data Analysis (EDA) notebook for the Kaggle competition:

**Titanic - Machine Learning from Disaster**

The notebook was originally created on Kaggle and contains my complete analysis workflow, including data exploration, visualization, data cleaning, and feature analysis.

The purpose of this project was to understand the Titanic dataset, explore patterns related to passenger survival, and practice the complete EDA process used in real-world data science projects.

---

## Dataset

The dataset is provided by Kaggle's Titanic competition.

It contains passenger information such as:

- Passenger details
- Demographics
- Ticket information
- Fare
- Family information
- Survival outcome

---

## Notebook

The complete analysis is available in:

`Titanic_EDA_Analysis.ipynb`

The notebook includes:

- Data exploration
- Data cleaning
- Missing value analysis
- Feature analysis
- Visualizations
- Observations and insights
